## 1. プロジェクト概要

| 項目 | 内容 |
|---|---|

| **目的** | 火災保険の適用可能性を簡易診断し、LINE登録へ誘導してリードを獲得する |

| **ターゲットデバイス** | SPファースト + PC対応（レスポンシブ） |
| **メインカラー** | オレンジ系（#E8721C を基調） |

#### スコア判定ロジック（フロントエンドで計算）

| 判定 | 条件 |
|---|---|
| **適用可能性：高い** | Q1が「持ち家（一軒家）」 かつ Q3〜Q9で「特にない」以外の選択が合計5個以上 |
| **適用可能性：ややあり** | Q1が「持ち家」系 かつ Q3〜Q9で「特にない」以外の選択が合計2〜4個 |
| **適用可能性：低め** | Q1が「賃貸」 または Q3〜Q9で「特にない」以外の選択が合計1個以下 |

※ 築年数（Q2）も加味：築10年以上で+1ポイントのような重み付けも可能

#### CTA
- 「今すぐ診断結果を見る」ボタン → LINE友だち追加URLへ遷移
- LINEアイコン + 「LINE登録して診断結果を受け取ってください」

## 7. 診断ロジック詳細

### スコア計算方式

```javascript
// 基本スコア = 0
let score = 0;

// Q1: 住居タイプによる重み
// 持ち家（一軒家）→ +3, 持ち家（マンション）→ +2, 賃貸 → +0, その他 → +1

// Q2: 築年数による重み
// 築0〜9年 → +0, 築10〜19年 → +1, 築20〜29年 → +2, 築30年以上 → +3

// Q3〜Q9: 各質問の「特にない」以外の選択1つにつき +1

// 判定
// score >= 8 → 適用可能性：高い
// score >= 4 → 適用可能性：ややあり
// score < 4  → 適用可能性：低め
```

### 分岐処理
- Q1で「賃貸」選択時 → 診断は継続するが、結果で「賃貸の場合は大家さんまたは管理会社にご確認ください」の注記を追加
- Q3〜Q9で全て「特にない」→ 「現時点では目立った被害はなさそうですが、見えない部分に被害がある場合もあります」
